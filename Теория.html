Ключевые слова let и const

let можно переназначит: let name = 3; name = 4 (корректно)

const переназначить нельзя: const pi = 3.14; pi = 4 (некорректно, js выдаст ошибку)
Если работать с объектом или массивом, то можно изменить массив/объект внутри константы, но нельзя присвоить новое значение константе
Сперва объявляется const, а если значение действительно нужно изменить, то ставится let

Функии-стрелки (arrow-функции)

Функция - это объект.
function square(x) { return x*x } - Это обычный вид функции
const = (x) => x*x; - Это стрелочная функция
у стрелочных функций нет свойства prototype

Параметры по умолчанию

function strange(count = 10; start = 0) { console.log ('lol', count, 'lel', start) } - в скобках параметры заданы со значениями по умолчанию

Rest параметр

function max(...numbers) { console.log(numbers) }
max(1,2,3,4,5,);
... - создаст реальный массив из numbers
rest параметр обязан быть последним в функции (в скобках). Нельзя иметь больше одного rest параметра.

Spread оператор

Spread раскладывает массив на список независимых элементов
... + array - это spread оператор.
const arr1 = [1,2,3]; const arr2 = [4,5,6]
const shadowCopy = [... arr1, ...arr2] - это копию массива arr1 + arr2;

Деструктуризация объекта

Деструктуризация позваляет достать значение из какой-либо структуры данных, массива или объекта
const person = { firstname: 'peter', lastname: 'smith', age:27};
Если мы хотим "вытащить" имя и фамилию и присвоить их константе, то:
const { firstname, lastname } = person; - то есть только что мы создали две константы и сохранили в них значение из аналогичных свойств person

const person = { name: { first: 'peter', last: 'smith' }, age: 27};
const { name: { first: firstname, last: lastname } } = person; - можно деструктурировать боле вложенные объекты и переписывать в те константы, которые мы сами задали

const person = { name: { first: 'peter', last: 'smith' }, age: 27};
Возможно указывать значения по умолчанию; const { role = 'user'}; То есть если в объекте Role нет, то он сделает его и поставит user.
